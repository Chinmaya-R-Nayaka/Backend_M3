# M3 Pediatric Clinical Data System

## Technology Stack
- Streamlit
- MongoDB Atlas
- PyMongo

## Modules Implemented
- Patient Management
- Growth Tracking
- Immunization Management
- Milestone Tracking
- Alert Generation

## Deployment
Deployed via Streamlit Cloud using GitHub integration.

## Environment Variables
Create .streamlit/secrets.toml and add:

MONGO_URI = "your_mongodb_connection_string"
